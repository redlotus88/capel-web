# capel-web
Capel实验室 Web项目

## 项目模板

本项目基于 [vue-typescript-admin-template](https://github.com/Armour/vue-typescript-admin-template) 项目开发。



## 项目技术栈

| 技术栈     | 版本   |
| ---------- | ------ |
| vue        | 2.6.10 |
| typescript | 3.6.2  |
| element ui | 2.12.0 |
|            |        |



## 项目环境

#### 编译

npm install

#### 开发环境运行

npm run serve

#### 启动端口

随机
